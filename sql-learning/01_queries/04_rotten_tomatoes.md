## 🎯 Core Concept
The `WHERE` clause filters data based on a condition, allowing you to retrieve only the rows that meet specific criteria.

## 📝 Key Syntax

### Basic WHERE Clause
```sql
SELECT column1, column2
FROM table_name
WHERE condition;
